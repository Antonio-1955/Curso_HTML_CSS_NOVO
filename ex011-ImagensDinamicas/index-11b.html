Outro modo de exibir áudios é detalhando o atributo source para o caso do navegador do usuário não ser compatível com algum formato de áudio. Primeiro vai tentar carregar o mp3, depois o ogg e por último o wav:

mp3 = mpeg - Formato com ótima compressão - Arquivos pequenos boa qualidade, são ideais para navegação

ogg = ogg - Formato com ótima compressão - Arquivos pequenos - boa qualidade, são ideais para navegação

wav = wav - Formato com baixa compressão - Arquivos muito grandes de excelente qualidade, mas ruim para navegação -
